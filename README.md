# Drool

Industry's leading AI coding agent Drool, open sourced by Udonge Open-Source Foundation (UOSF 😭💢).

## Install

### macOS and Linux

```sh
curl -fsSL https://raw.githubusercontent.com/udonge-foundation/drool/master/install.sh | sh
```

### Windows PowerShell

```powershell
irm https://raw.githubusercontent.com/udonge-foundation/drool/master/install.ps1 | iex
```

Both installers download the latest release binary and install `drool` to `~/.local/bin`.
